# SKILLS - Raiz del proyecto

Skills base obligatorios para cualquier fase:
- agent-governance
- refactor
- security-best-practices
- pytest-coverage
- create-implementation-plan

Plugins priorizados:
- Browser
- GitHub (cuando aplique)
- Spreadsheets (validacion XLSX)

Regla operativa:
- Antes de codificar: usar enfoque explorer para analisis breve.
- Durante implementacion: usar enfoque worker con ownership explicito por capa.
- Cierre: pruebas + evidencia + actualizacion documental.
- Todo comando de ejecucion local debe canalizarse via `pnpm` desde la raiz.
